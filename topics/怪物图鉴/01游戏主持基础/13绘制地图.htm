绘制地图 Drawing Maps

一旦你有了主意，确认了冒险的范围和挑战，下一步就是设计冒险地点，无论它是地下城，沙漠废墟还是海底珊瑚森林。

如果你为自己缺乏创造地图的艺术能力而担忧，别这么想！你的地图从根本上应该为目的服务，构建出行动和故事的流向框架，允许你跟踪战役的进程和可能性。没有人期望你作出令人惊叹的，看起来就像出版模组那样专业的地图。只需遵循几个最佳步骤，就能让你的地图既有趣又清晰。

所以，拿出以下用品然后开始。

l   几张方格纸

l   草稿纸

l   铅笔，最好是H或3H，这样留下较浅的印记容易被擦掉

l   橡皮

l   墨水笔（可选）

l   标尺和指南针（可选）

在制图过程中的任何阶段都不要因犯错而沮丧。你随时可以修正它们——橡皮擦是你的朋友！画图或擦除过多会让纸变得脆弱，但你始终可以扔掉草稿并重新开始。提醒自己，这不必是有史以来最精美的艺术品，它只不过是一种让故事保持直观的工具。

创作图例 Create a Legend

首先，列出你的冒险地点应包含的要素——从地理上的地形到建筑物中的房间类型，再到玩家角色将面对的怪物和陷阱——然后确定地图的比例和方向。你用的比例，方向和用来表示要素的符号的说明列表，都将出现在地图的图例中。

通常冒险地图的比例最容易决定。如果该地图用于在1英寸等于5英尺的方格上进行战斗，则可以假定方格纸每格为5英尺。如果你要绘制更大的区域，就用每个方格代表10英尺甚至更大。

对于方向，向北画一个箭头，在该箭头的上方写一个“ N”，这就完成了。

地图描绘的各种细节的特征很难预测。决定地图应包括的要素以后（例如区域地图的怪物、秘密门和地城的陷阱、或住所和遗址），请考虑一个符号来代表地图上的各个要素。你也可以使用样图中的符号。

开始草图 Start Sketching

既然有了图例，是时候开始绘制地图了。拿起铅笔，开始绘制冒险地点的草图。

要绘制地城建筑群，就先详细绘制走廊和房间。要绘制室外区域，请先从水体，河流和地形开始。不管要绘制什么内容，都可以用较浅的线条进行绘制，方便擦掉——因为你可能犯错，或之后又有了更好的主意。随着地图逐渐成形，就可以开始用图例里列表中的符号所对应的要素来充实地图。

不要使你的地图过于复杂，尽可能简化你的地图，这样做可以突出重要的细节，有助提高地图的可读性。

在绘制草图时——尤其是在绘制荒野或建筑物时——从现实世界中汲取灵感。上网搜索能找到参考图片，当你决定在哪放置树木，峡谷和溪流的形状，或复制古代，中世纪，文艺复兴时期的建筑，或幻想建筑时，这些参考图片会有所帮助。将这些参考作为地形和建筑的基础，能使你的创作看起来更加自然和可信。

你绘制地城和类似的冒险地点时，请记住居住在建筑群中的生物的体型，并为其提供足够的生活和移动的空间。换句话说，请确保古黑龙能够穿过入口到其巢穴！其次，避免空房间，不要仅仅为了想有一个房间而放房间。每个房间应该有一个用途。就算已经废弃不用了，每次你创造一个房间时，都要问自己：“为什么要建造这个房间？”最后，避免使用对称或直接的无聊结构。探索的乐趣之一就是惊讶于所发现的东西，而不必要的对称性和重复的布局可能会破坏这一乐趣。

编号遭遇区域 Number Encounter Areas

下一步是对遭遇区域进行编号。你可以之后在遭遇记录中使用这个编号系统。如果要设计多个地点或大型地城的不同层，可以考虑在编号之前添加字母名称。冒险地点的第一个区域可以是A1，而下个冒险地点的第一个区域可以是B1，依此类推。

增加清晰度 Add Definition

在绘制草图并作好图例之后，你可以用笔迹更深的铅笔或墨水来巩固草图。这使你可以保存地图，以便重新进行这一冒险或在其他冒险中重复利用。如果你有艺术灵感，则可以使用彩色铅笔，记号笔或颜料为地图上色。
