奇术师


 关于译名，我给出一点自己的理解:
奇术师的译名引起了一些争议，我来解释一下。
奇术师是形似施法者又不是施法者的奇迹创造者（20级专长Wonder Worker），他所使用这种古怪的超自然力量用奇术来表达是比较合理的。
奇术师虽然使用了大量的道具，包括主要的奇具，但这些和一般我们认为的具有真实力量能强化自身的道具不同，奇具和秘传的力量来自于奇术师赋予的象征意义。奇术的本质就是象征主义的运用，用某种实物象征一种概念，再用这种概念击倒敌人。
这点在秘传相克能力中提到的自定义弱点的例子就能看出，对付暴君，于是使用解放俘虏的镣铐。这是十分不讲道理的做法，但奇术师能将其实现，就是利用了破损镣铐中象征的自由的概念去克制暴君。这种做法比逻辑严谨的学者，奇术师更像是一个艺术家。这也能解释为什么奇术师的主属性是魅力。
也就是说，奇术师运用的这些奇术，虽然存在道具，但并非以道具为核心，甚至这些道具在其他人手里一文不值。就如魔术师的许多小道具一样，魔术师的创造力和手艺才是核心，这些小道具都只是辅助魔术师表演魔术的工具而已。奇术师的道具，是奇术师借此来象征概念以施展奇术的实物而已，而奇具，就是自身人格力量的象征。基于这种考虑，我认为不应该在职业名称中强调道具。

世界充满了不可解之物：古代魔法、已逝神祗、还有一些更捉摸不清的东西。为了应对它们，你挖掘每个魔法根源中的最精华的部分，构建了一套秘藏——这里放着破损的圣物，那里摆着榭寄生的嫩枝——你能借此发掘任何生物的弱点，并最终战胜它们。你携带的神秘奇具既是徽章也是武器，它所象征的重要性能帮助你协调并征服超自然。通往力量的每条路径都伴随着限制和代价，但你能将其全部化作自己的优势。你是奇术师，你创造奇迹。

在战斗遭遇中…
你使用你的奇具和由多种神秘道具和诡计构成的武装库在战斗中进行支援，总是能够展示出对敌人来说正确的克星并利用其弱点，或从超自然的力量中保护盟友。

在社交遭遇中…
你理解并操作人与人之间的社交联系，十分类似于你在神话和概念之间努力操作的那种更大的纽带。这能帮助你在世俗界和超自然界之间协调时找到彼此的共同基础，或甚至站在其中一方的立场上对抗另一方。

在探索中…
你调查周遭不可解的现象，采取合适的预防措施从超常的威胁中保护你和你的盟友。必要之时，你能创造你自己的道路穿过未知。

在休整期…
你更深入地研究你在上一次冒险中发现的超自然神秘现象，以预测它们会对未来造成什么样的影响。你重新使用废墟中的遗物、培植药草、或铸造一些小装饰物，以在你遭遇到每一个新的威胁之后更新你收集的超自然工具。你向其他人、其他奇异生物、或神秘势力呼唤恩惠并加以回报。

你可能…
  • 总是为最糟糕的情况做好准备；令人印象深刻的是，你在面对命运的捉弄时总能处理好任何事情。
  • 假定万事万物之间都具有某种深层的联系，而非那些你只能在表面上看到的现象。
  • 痴迷于某种超自然现象，试图把每件事情与其联系在一起。
其他人可能…
  • 当一个超自然威胁出现时，向你学习它的弱点。即使你的解释是现场发明出来的，它们也似乎是有效的。
  • 在状况似乎陷入绝望之时，依靠你给出解决办法。
  • 将你误认为一个普通的法师或魔术师。

关键术语 Key Terms
你将会在许多奇术师职业特性中看到这些术语。
秘藏Esoterica：许多奇术师专长和职业特性中都展示了秘藏特征，这些能力都涉及到你随身携带的各种护符、超自然饰物、以及其他物品。具有秘藏特征的能力要求你拥有你的秘藏以使用它们。通常都会假定你总是随身带着你的秘藏，但在某些稀有的状况中，你可能没法拿到它，或者你的装备被从你身上剥夺。
临时物品Temporary Items：一些专长能让你准备临时物品。十分类似炼金术士创造的灌注物品，临时物品只能保持一小段时间，之后就会失去效用。临时物品的质量显然不如其他物品，通常无法出售。如果一个能力没有列出临时物品能够保持的时间，这个物品保持到你进行你下一次每日准备时。任何由临时物品创造的效果如果在此时还未终止，则也同时终止（除非这是一个永久效果）。

秘传学识 Esoteric Lore
你对未知事物的体验，以及你与其他奇术师交流的传说，将关于每一种异常现象的知识传授于你。你在秘传学识Esoteric Lore上受训，这是一种特殊的学识技能，可以用于在作祟、诅咒、以及任何种类的生物相关的主题上进行回忆知识Recall Knowledge，但这个回忆知识Recall Knowledge无法用于其他主题。和通常的学识技能不同的是，你在秘传学识Esoteric Lore的检定上使用魅力作为你的调整值。你也获得诡异知识Dubious Knowledge技能专长。
在3级，你成为秘传学识Esoteric Lore的专家；在7级，你成为秘传学识Esoteric Lore的大师；在15级，你成为秘传学识Esoteric Lore的传奇。

第一奇具和秘藏 First Implement and Esoterica
你的奇具是一种具有象征意义的特殊物品：当你与超自然打交道时，这是你的徽章；而当事态转向暴力，这则是你强大的工具。从后文的奇具描述给出的选项中选择一个奇具（符Amulet、铃Bell、杯Chalice、灯Lantern、镜Mirror、帜Regalia、典Tome、杖Wand、兵Weapon）。你开始游戏时会携带该类型的一件平凡的物品，而且你获得该奇具的初始增益。虽然该奇具对你来说十分有用，但如果要出售它，它通常都没有价值。如果你获得了和奇具类型相同的一件新的物品，你可以花费1天的休整期来处理该物品，以将你的奇具切换成这件新物品。当你作为奇术师得到成长，你将会收集到最多3件奇具，并解锁储存在每件奇具中的隐藏潜能，这样你就可以混合并配对它们的增益以与你的状况相适应。
虽然你的奇具是你与超自然打交道时的主要工具，你也知道时刻做好万全准备是一件有价值的事情。你持续收集并携带着多种较小的神秘物品、一些具有超常联系的材料、和一些用于民间习俗的物品：这就是你的秘藏。这可能包含了寒铁钉、卷轴和雕像的残片、据称来自一位圣徒的骨片、以及其他类似的物品。你在自己身上一个容易拿取的地方保管着你的秘藏，并能熟练地同时挥舞你的奇具和秘藏，因此你可以在一只手持用一件奇具的同时，用相同的手取出并使用秘藏。

弱点发掘Exploit Vulnerability
你知道每一个生物，无论有多么难以理解，都存在着弱点。识别正确的物品，并为其赋予力量，你甚至可以击倒最顽强的怪物。
你获得弱点发掘Exploit Vulnerability动作。

弱点发掘Exploit Vulnerability[[A]]
秘藏 操作 奇术师
频率 每轮1次
需求 你正持握着你的奇具。
你遍历你的经验和学问，以识别某件可能能反制你的对手的物品。你从你的秘藏中找到一件具有合适的超自然特质的物品，然后使用你的奇具将其中残余的力量烧成熊熊烈火。选择一个你可以看见的生物，并尝试一次秘传学识Esoteric Lore检定，对抗该生物等级的标准DC，此时你从秘藏中找出了合适的物品，并使用你的奇具给予它力量。你获得如下效果，直到你再次使用弱点发掘Exploit Vulnerability。
大成功 你记得该生物的弱点，且随着你为你的秘藏赋能，你灵光一现，甚至获得了关于该生物更多的知识。你了解到了该生物全部的抗力、弱点、以及免疫，包括抗力和弱点的总数，以及任何不常见的脆弱之处，例如用什么法术可以穿透一个魔像的反魔法能力。你可以发掘该生物的凡体弱点Mortal Weakness，或个体克制Personal Antithesis（见后文）。你对该生物进行的无武装和武器打击Strike如果还不是魔法的，则也变成魔法的。
成功 你回忆起了关于该生物的重要事实，了解到了它最高的弱点（或其中一个最高的弱点，如果它有多个相同数值的最高弱点），但不知道它其他的弱点、抗力、或免疫。你可以开发该生物的凡体弱点或个体克制。你对该生物进行的无武装和武器打击Strike如果还不是魔法的，则也变成魔法的。
失败 你无法回忆起该生物的显著弱点，你改而尝试发掘一个更个人化的弱点。你只能发掘该生物的个体克制。你对该生物进行的无武装和武器打击Strike如果还不是魔法的，则也变成魔法的。
大失败 你无法记起应该使用的正确物品，并在你慌忙翻找秘藏的时候变得心烦意乱。直到你下一回合开始，你陷入措手不及flat-footed。

你可以以两种方式之一来进行弱点发掘：要么引入一种可以反制该类型生物的性质，要么利用你的秘藏尝试一种更加即兴、特设的方法，能对任何生物强加一种定制的弱点，尽管这通常不像生物已存在的弱点那样可怕。
凡体弱点 Mortal Weakness 识别一个生物的弱点之后，你使用一些可以和该主题产生共鸣的秘藏，调校你的攻击，使其与你的发现和谐统一。你的无武装和武器攻击会触发你通过弱点发掘Exploit Vulnerability发现的最高弱点，即使你武器造成的伤害类型没有改变。该伤害能影响你弱点发掘Exploit Vulnerability的目标，以及与该生物类型精确相同的任何其他生物，但对其他拥有相同弱点的生物无效。例如，当与一群狼人战斗时，你可以使用银片或破损的月长石，在你对任何狼人进行的攻击上造成可以触发银弱点的伤害，但你无法将这种伤害应用在任何其他具有银弱点的怪物上。
个体克制 Personal Antithesis 通过强制展示一份秘藏并为其赋能，你对一个生物临时客制了一种弱点；例如，对抗一个暴君，你可以取出一条曾束缚俘虏的破损锁链。这导致目标生物，也只有目标生物，获得对你的无武装和武器打击Strike生效的弱点，数值为2+你等级的一半。

奇具赋能 Implement's Empowerment
你奇具的力量也能实现更常见的战斗任务，它的力量会加入并增幅符文和其他魔法赋能的效果。当你打击Strike，你能够使用一件你正持握的奇具描绘一种神秘的图纹以强化这次打击Strike，让它造成每武器伤害骰2点额外伤害。引导这种能量需要完全使用你的双手。如果你在任何一只手上持握着除了单一把单手武器、其他奇具、或秘藏之外的东西，你就无法获得奇具赋能的好处，而且你必须持握着至少一把奇具才能获得该好处。
