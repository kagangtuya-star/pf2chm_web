运行团期

运行团期 Running a Game Session

一个战役发生在一系列的团期中。每个团期长度通常有数小时，伴随数个遭遇、一些探索，还可能有休整期。你可以把一次团期当做电视连续剧的一集，其中应该有些转折、转变、改变还有结尾，让观众期待接下来会发生什么。

安排团期 Planning a Session

跑团最大的挑战之一，就是敲定一个大家都能聚在一起跑团的时间。如果你是GM，通常这个责任会落在你身上，因为你就是那个要在团期与团期之间准备游戏的人。许多游戏都有一套行程表，例如一周一次、双周一次或一个月一次。你的团聚会的频率越低，你会需要更好的笔记还有前情提要让每个人进入状态。
　　安排一个所有人都可以到场的时间，也为游戏开始设定一个时间。这可以让所有人停止闲聊、快点赶来跑团，吃东西变的快点，好让你可以开始游戏。在自己的心理定下一个结束时间也很重要。通常一个团期会持续4小时，但也有两小时或马拉松式的团期。少于两小时通常在《探索者》的战役中做不了什么。如果你的团期长于2小时，要规划约15分钟的休息时间（但上厕所跟倒饮料不算在内，玩家想去的时候就该让他们去）。

开始团期 Starting a Session

当所有人就位，把大家的注意力集中起来，接着开始下面的话题。这边提供大致的顺序，你可以根据你们的风格或团期的需求去调整。
  • 先简扼的提一下之前的团期发生了什么事。
  • 确定角色在本次团期开始时人在哪里。他们在上次最后的挑战后休息了吗?他们在走道上，准备去劫掠地城的下个房间?告诉玩家他们上次跑团结束后，他们的角色是否有时间可以休息或恢复。
  • 提醒玩家他们在每次团期开始时都有1点英雄点（P.507）。
  • 设定目标。玩家必须要对下一步的行动有所想法。重新确立玩家团队早就有的目标，接着让玩家自己衡量这目标是不是还适用，以及他们是否希望在本次团期中完成什么其他的任务。
  • 开始冒险!选择你要从哪个游戏模式开始。你可以询问和某位角色相关的问题、有怪物来袭，让所有人立刻骰先攻或简单的描述一下玩家角色周遭的环境以及对环境的感受，让角色可与之互动。

跑团 Running a Session

在跑团中，你要负责维持游戏的运做、管理不同的游戏模式、场地的问题以及规则的判定。你也需要注意一下大概的时间，好能在让大家都感觉舒服的情况下结束游戏。
　　你是规则和由你与其他玩家所共享的幻想世界中的接口。玩家会询问你问题，并基于自身的假设开始行动。这个世界的真实取决于你，但并不需要单方面做这件事。你会从设定的背景故事、你的准备还有其他玩家在桌上的意见与假设里获取消息。注意，直到你宣告了什么，你的计划永远都可以改变。举个例子，如果你起初是让旅店的所有者是个善良的好人，但玩家误解了她，并且对她的意图虚构出一套听上去很有趣的阴谋论，你可以把旅店的所有者变成一个邪恶的爪牙。
　　你还可以决定角色和敌人什么时候该进行检定，以及她们骰出来的结果。这比较常出现在遭遇之外，因为遭遇有严格控管，什么时候要检定，以及如何判断。在遭遇中，玩家通常可以决定角色的回合，而你的附和只是说这下攻击有没有中，或当前环境下是否有什么东西需要角色去进行检定。

聚光灯 The Spotlight

在游戏进行中，去观察谁在聚光灯下。我们很容易把注意力放在突出的玩家或角色身上，但你需要把注意力放到所有的玩家上。如果有玩家有段时间没有出什么力，停下并询问她：你的角色现在在做些什么?如果玩家并不确定，为这个场景增添一些玩家会感到兴趣的细节或是非玩家角色。

分心与干扰 Distractions and Interruptions

当大家同在一起时，你要聚拢玩家的注意力确保游戏的退近，并引领大家抵达值得纪念的时刻。但有很多干扰会打破这个流畅。如果是度还能接受。如果分心的太频繁就会出问题，因为分心会让玩家错过团里的东西，因为对团的内容分离，做出误判。但所有游戏都有休息时间和离题时间，有些团是刻意的，有些则否。为你的团在消遣上寻找正确的平衡有其必要性。
　　这个游戏是社交聚会，所以一定有个场合可以用来交流并且和游戏没有直接关连。但如果这类的干扰太频繁，或有些人对其他人说话说的太过分，也会出问题。如果有玩家屡次打断你和其他玩家，或每次都在游戏的关键时刻说笑话泼冷水，你要告诉他们，请在适当的时间点才表达意见。通常，你只需要举起你的手或用其他的方式跟该玩家宣告，他说话不按顺序已经拖延到大家了，请他直到你与其他人说完话后再说。
　　手机或是其他行动装置是另一个分心来源。完全禁止这些东西不太可能，有许多玩家要使用APP骰骰子、管理角色卡、他们需要看些文件好回答角色的伙伴的问题、检查工作计划或跟其他倚赖他们的人保持联系。但你可以设定一套基本规则，禁止在非敏感时刻或跟游戏无关的时候使用行动装置，像是刷新设交媒体、察看曲棍球比赛的成绩、玩手机游戏或是回答并不急迫的内容。当玩家的角色在「幕后」时，你可以放宽规则。如果玩家的角色并不在场，对玩家来说这是个使用行动装置的好时间。

裁决规则 Adjudicating the Rules

做为GM，你有责任去解答所有对规则的争论。记住，让你的游戏继续下去比永远100%正确还重要。在游戏中查规则会让游戏速度变慢，所以在许多情况下，做出你最好的猜测会比快速翻书查准确规则还好。（在休息时或团期后去查规则比较好!）
　　想要快速做出决定，你可以使用下面的指引，他们也都基于本游戏规则原理。你可以把这些指引跟DC指引（P.503）印出来做为快速参考。
  • 如果你不知道一个迅速的任务要花多少动作，那就算一个动作，如果角色不应该能在一轮中进行三次的，就算两个动作。
  • 如果你不确定这个任务要使用什么动作，参考与之最相似的基础动作。如果你没有找到，虚构一个未定义的动作（P.XXX），并增添所有必须的特征（通常事攻击、专注、操作或移动）。
  • 当双方开始对抗，让一边骰骰子去对抗另一边的DC。不要让双方都骰（先攻是个例外）。掷骰的角色通常是主动方（骰豁免例外）。
  • 如果有个效果提高或降低了成功的机会，给予+1环境加值或-1环境罚值。
  • 如果你不确定某个特定挑战应该多难，使用团队等级的DC。
  • 如果你虚构一个效果，那生物应该要在大成功的时候失能或被杀（如果是过豁免，那就是大失败）。
  • 如果你不知道该用什么检定，挑个最适合的技能。如果没有可以用来进行回忆知识检定的技能，那可以使用适当的学识技能（通常是未受训熟练度）。
  • 用角色的每日准备做为时间依据，来重设任何会持续约一天的事物。
  • 当玩家完成值得纪念的事蹟，却又没有给XP的规则，奖励他们一个成就的XP（10-30XP，详情见P.507）。
  • 当玩家在任务中失败，寻找一条失败后继续前进的路，让故事以负面后果继续推进，而不是因为失败让过程完全停止。

非团期游戏 Off-session Gaming

整团一起进行团期并非是《探索者》唯一的玩法。找机会在游戏外拓展惯例行程，可以让你的团在团期之间保持活跃。
　　你可以和一个玩家未他的角色一起跑个迷你团期，做一个对故事很重要，但跟团里其他人无关的任务。你和玩家可以在一段固定的休整期，透过电子邮件或聊天软体做这件事。你也可以给玩家机会对故事的细节进行协做，例如让玩家位冒险中的团队设计一个纹章，或标示他们的大本营。你甚至可以在下次团期，因着在团期外的事件，奖励玩家英雄点。
　　有些事件不适合在团期外进行。强烈影响角色而他的玩家又不在现场的所有事件，应该先压着，直到所有人都能参加。对所有角色概述一下团期外发生的事件十分有帮助，能让其他人不会感觉到自己的排挤或感到失落。

责任分摊 Sharing Responsibility

就因为妳是GM并是名面上的负责人，并不代表妳必须把所有额外工作都做了，好让战役能运行。下面有些任务，例如排定游戏的行程表、记笔记以及作前情提要可以分配给其他玩家。妳也可以让某人帮妳记录遭遇中的先攻顺序或敌人的生命值，或如果团队够大，妳甚至可以让某个想控制敌人胜于控制自己角色的玩家去控制敌人。如果有人能带团、为大家提供零食或是承担不直接与游戏相关的责任，那也很棒。
　　最好在游戏一开始就制订好责任分担表。问玩家他们会想要承担哪些责任。如果妳在战役过程中开始感到不知所措，妳可以重启此话题，试着找出新的选项，直到妳觉得合适为止。

Paizo出版的冒险 Paizo's Published Adventures

妳可以在Paizo.com、妳家附近的游戏商店或书店到下列的冒险产品。如果你想要某条产品线上的所有冒险产品，妳可以在Paizo.com上进行订阅。

探索者冒险之路 Pathfinder Adventure Paths

探索者冒险之路是一个大型且多卷的故事，每月出版一卷推进。每个冒险之路的第一卷通常会从1级开始，同时每一卷都是自成一体的故事，最后将在最终卷推向整个故事的高峰。每一卷通常会有新怪物、新规则还有新的世界设定细节。每个冒险之路都有不同的主题，并且设定横跨内海内外。

探索者冒险 Pathfinder Adventures

探索者冒险是涵盖多种等级区间的独立冒险。他们自成一体通常有其独特的体裁与主题。妳可以独自使用探索者冒险之路，或是将它做为正在进行的战役的一部分，对于有着相似主题的冒险路来说，会成为不错的支线冒险。

探索者协会剧本 Pathfinder Society Scenarios

探索者协会剧本是探索者协会角色扮演指南使用的冒险，妳可以在进行探索者协会实使用他们，也可以为自己的剧本而用。这些剧本通常要跑4到5个小时，所以妳可以在短时间内带完一个完整的故事，但他们也是一个更大剧本的部分，可以合并这些剧本，形成一个长战役的基础。

特殊环境 Special Circumstances

团里的角色有时会进行比规则或冒险规定，可你认为应该更简单或困难的任务，例如有个角色在他的家乡进行蒐集情报。这种情况下，你可以使用环境加值或罚值。一般来说，在一个影响较小但特定的环境会有+1或-1，但如果影响较大，你可以调整到+2或-2。加值或罚值的极值（+4或-4）只应该出现在某人具有压倒性的优势或他想尝试近乎绝望但又不是不可能的事情。
　　你可以为行动添加特征。这么说吧，席拉把她的剑探入煤炉后对着有火焰弱点的敌人挥了过去。你就可以为这次攻击添加火焰特征。如果角色想要获得这种优势，通常要使用一个动作才行，所以席拉会在一次攻击中获得这项益处，但想如法炮制，她需要再把剑伸进煤炉烧。

合并额外选项 Incorporating Additional Options

你可以让玩家使用额外规则或角色选项。如果你有信心让角色选择这些特殊选项会提升你游戏的体验，那就放手做吧!如果你不确定或是对这要求感到忧虑，你可以不允许，由你决定。但是，试着跟玩家取得共识或以另一种方式暗示。如果你想是用这个选项但又担心它在之后会成为问题，先跟妳的玩家说明并解释你只是战时允许这个选项，但有可能在日后改变心意，取决于在之后的游戏里这个选项的使用状况。
