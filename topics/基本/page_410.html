真名

真名 True Names

真名魔法的本质基于这么一个原则：如果你知道了某人的名字，你就拥有了控制他们的权利。毕竟，命名一件事物就是在描述这件事物——但如果要准确地描述一件事物，你就必须理解它，而一旦你理解了它，你就可以操控它。
稀有度：稀有
　　一件事物越是古老、越是复杂，那它就越是有可能拥有不止一个名字。一个人会有很多名字：他们继承的名字、他们出生时取的名字、他们自己选择的名字、别人赠与的绰号的名字。然而，真名魔法则认为存在有一个最终的、秘密的名字——一个无人知晓亦或是鲜为人知的真名——它能够代表我们所有的复杂性。
　　真名可能是在宇宙经由自然过程被创造生成时由诸神所指定的，并且就如同指纹一般独一无二。大多数人从未知晓、甚至可能从未意识到他们拥有这么一个名字，但是真名魔法的大师们花费了无数的时间来研究这些名字，把它们记录在长长的名单上，使用它们来召唤、命令、放逐亦或是支配他们的敌人。

名字的本质 The Nature of Names

根据真名魔法的用途，名字被划分为三类：公众名、私人名、真名。公众名是当我们没有一个更具体的名称时用来称呼某类事物的名字。它们通常仅有一个词——比如“草”或“长颈鹿”——但它们也同样包括别称和昵称。一个公众名字可以在未经准许的情况下使用，这也是依赖名字的施法者对那些他们不认识的人施加魔法的方式，即兴创作一个绰号或是简单的用一个名词来指代目标，比如说“精灵”。
　　私人名包括了大部分的出生名、赋予名和选择名。它们并不一定是秘密，但同时也并不是每个人都知晓它们。了解这类私密之名意味着你可以更好地理解和影响这个个体。地点、动物和其他物体可以由个人或是群体赋予其私人名。一座山就只是一座山，然而附近的人们将其视为守护女神并开始称其为“石母”，那座山便有了一个只有这个群体才知晓的私人名。
　　一个人、地点、或事物也许会有许多公众名或私人名，但它只能有一个真名，一个完美地诠释了其本质的名字。根据他文化的不同，有的人可能甚至都无法意识到自己拥有真名。在一个不知道真名的社会中长大的孩子，可能一辈子都只依靠着私人名在生活，甚至从未怀疑过自己是否还会有一个真名，就更别提知晓其名为何了。但是在魔法被泛用的文化中，许多人都知晓自身的真名，并且会采取措施来保护它。他们保守着真名的秘密，只会透露给最为信赖的挚爱。一个人最早得知真名的方式，可能是由一位精研魔法之人通过调查研究寻得其真名后告知他本人。但在其他的一些社会中，人们仅有其灵魂伴侣赋予真名，而其灵魂伴侣也无需经由他人之口即可知晓此名。知晓自己的真名能够让你对自己有一个更为深刻与内省的洞见，让你能够理解自身的动机和心理，帮助你达成自我实现，避免分裂和混乱。
　　按照释义，真名概括了一个人从过去到现在的一切。它们是一个人的本源内核，这也意味着它们无法改变。然而生命、魔法以及世界皆为神秘莫测之物！有那么一些个体，他们极具转变性的经历使得他们在故事前后成为了完全不同的人，而当他们转变之时，他们的真名也随之改变。

无名之物 Nameless Things

有时，一个异怪、天族、魔族、观察者、魂魄、或者其他生物来自于另一个位面或是另一个全然不同的宇宙——一处拥有不同法则和魔法的存在。如果这个生物所来之处本无真名存在、亦或无真名可选，那它也许就根本没有真名。博学士们将这些生物成为无名之物，并且没有人能够确定他们是否真的存在；一些学者认为，一旦某个无名的事物进入了这个宇宙，那它就会像其他所有活物那样被赋予名字。不过，如果你的游戏中确实存在无名之物，赋予他们如下稀有能力。
　　无名之物:此生物没有真名。它无法被任何在距离上无需效果线的魔法效果选为目标，例如探知术和短讯术。它在对抗其他魔法效果的豁免上获得+2环境加值。

学习真名 Learning True Names

可以通过许多方式来发现和学习真名。少数稀有的秘密社团可能会拥有“指名者”，他们的唯一工作就是通过死记硬背和反复练习将冗长的真名列表传授给社团的追随者。但对大多数人而言，对真名的发现往往是广泛研究的结果——尽管这种研究所要求的形式是如此的难以预测。
　　真名有时会被发现记录在某个早已作古的施法者的个人日记或魔典中。那些以对抗某类生物而闻名的组织会在偶有揭示出敌人的真名后会将其编撰成册。天使、恶魔、以及其他类似生物的真名会极其罕见地在为其奉献的祈词或是为了保护不受其伤害的颂唱中被发现。秘学士们有时会通过深度的冥想出神——很有可能是在迷幻药物的帮助下——让他们的心智穿越宇宙，通过奇异的顿悟获取一个真名。那些侍奉并且理解自然的人同样知晓有关原能和第一世界的存在的真名早已被编入了这个世界的本身——在树木的年轮中、在地质的断层中、在地上积雪的图案中——仅仅只是等待着那些知晓所寻之物为何的人来破译。对于少数的幸运儿而言，作为寻得了灵魂伴侣的象征，真名自然而然地就来到了他们身边。所有这些信息的范例都能通过使用研究子系统（GMG第154-155页）来实现。

使用研究子系统 Using The Research Subsystem

当寻找一个特定个体的真名是故事的关键，并且时间有限时，便可使用研究子系统。将图书馆等级设置为被PC们寻求真名的那个生物的等级。考虑一下你会在故事中使用哪一类怪诞奇幻的图书馆。例如，想象一下玩家角色们正在寻找一只菲斯魔（也被成为契约魔）的真名，为了将一个NPC从地狱契约的苦果中拯救出来。你也许可以在切利亚斯或是某些其他普及有魔鬼知识的地区设定一个地狱图书馆，或者干脆就放在地狱里面！这样的图书馆里可能有着大量的守卫和陷阱，但也同样有可能从那里各种各样的契约中了解到更多的真名。

简易名号研究 Simplified Name Research

有时候GM并没有时间来创建图书馆或是使用研究子系统，但仍希望PC能基于故事层面的原因研究某个特殊存在的名字。在这种情况下，他们可以使用下列的获知名字休整期行动。因为得知一个真名意味着将使该生物任由指名者摆布，在默认情况下该行动无法揭露真名，但可能会带来指向一个生物真名的线索。

你花费一个星期来试图揭露和获取一个生物的名字。你尝试的具体形式取决于你使用的技能、能够调用的资源以及其他的各种情况。决定你是搜寻某个特定个体的名字，还是些宽泛的名字。如果你寻找一个个体的名字，你必须能够清晰地识别出那个个体；例如，“指挥入侵的将军”是足够了，但是在你不知道是谁谋害了公爵夫人的情况下，“杀害公爵夫人的人”就不行了。如果你想在宽泛的范围内搜寻名字，则指名出一种生物类别。
　　由GM来选择DC，通常会基于所询问的生物的等级。如果你在宽泛地搜寻名字，DC一般基于GM决定提供的生物的等级，通常是所选类型里与你等级相同或是更低等级的生物。GM可能会根据你可调用的资源、使用一个特别合适或不合适的技能、或是其他情况来调整任务的DC。尝试使用一个能够回忆知识的与该生物相关的技能来进行检定。在尝试获取名字之后，你通常无法再次尝试对同一个个体使用获取名字，除非你在DM的判断下获取了大量新的信息源。

成功 如同大成功，只是你只能发现一个私人名，并且无法找到真名破片。
大失败 如果你正在搜寻某个特定个体的名字，那么你无法发现任何新信息，并且该个体会察觉到你的尝试。如果你在宽泛地搜寻某个特定种类中的名字，你所发现的生物的名字可能会给你带来麻烦，也可能一个完全不同种类的生物的名字。

使用真名 Using True Names

某些法术、专长和物品具有真名特征。该特征意味着它们要求你得知一个生物的真名后才能使用它们。不过即使没有这些特殊能力，得知一个生物的真名依旧能够给你带来一定的优势。如果知道一个生物的真名，你能够在下列检定中获得+2环境加值。
  • 关于该生物的回忆知识检定
  • 涉及到该生物的奥法、自然、神秘、宗教检定，例如解译（它的）文书、辨识（它的）魔法、学习（它的）法术
  • 直接用于该生物或与其相关的欺骗、交涉、威吓检定，例如胁迫它、搜集（它的）信息、或是乔装成它

破碎之名 Fragmentary Names

有时角色只知晓一个生物真名的一部分，而非全部名字。如果你只知晓名字的破片，你无法获得使用真名所罗列的好处，但你仍能使用具有真名特征的能力，尽管这是异常危险的行径。如果你的真名能力要求你尝试一次检定，将你的成功度降低一阶。如果你的能力要求被指名的生物尝试一次豁免骰，将他们的成功度提升一阶。例如，唤用真名将无事发生，因为其效果将被取消。如果真名能力同时具有失能特征，并且该生物的等级也足够高，那么这些效果将累加在一起，将你的成功度降低二阶或将他们的成功度提升二阶。
