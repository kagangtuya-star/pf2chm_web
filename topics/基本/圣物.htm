圣物 RELICS
嘘！来这里！老板没看见你吧？好吧。这个宝库很特别，很隐秘，很神奇！这个宝库里的东西不仅仅是宝藏，还有很多故事。这里的每件东西都是某个大英雄使用过的，他们做了很了不起的事情，创造了自己的传奇，每件物品都是传奇的一部分。其中一些甚至是多个传说的一部分，从一个英雄传承给另一个。如果我离开这里去冒险，我希望这样的传奇圣物也能成为我故事的一部分！
——阿纯

新性相 New Aspects
性相对圣物来说可能是各种各样的、能体现圣物的存在和魔法的力量。本节在其他书的基础上（游戏主持人指南94–105页，魔法之秘230–231页）增加了如下新性相。


艺术赋礼 ARTISTRY GIFTS
艺术性相的赋礼能提高你的艺术表现力，让你留下更加深刻的印象。

情绪赋礼 EMOTION GIFTS
拥有情绪性相的圣物体现了某种强烈的情感，从愤怒、惊叹、奉献、恐惧、仇恨、喜悦、爱、苦痛、骄傲或悔恨中选择。这些赋礼有时和魔法之秘中的宣情魔法状态相关，但不使用情绪焦点。作为替代，如果赋礼基于奉献或爱，它的情绪焦点为你的盟友之一，由你在启动效果时选择；如果赋礼基于仇恨，它的情绪焦点为你的敌人之一，同样由你在启动效果时选择。

智能圣物 Intelligent Relics
任何智能物品都可以成为圣物， 圣物也可以被启迪智慧，如果你打算将其添加至你的战役中，下面的赋礼能够促成这种情况发生。

智能圣物 INTELLIGENT RELIC       下等赋礼
[智能]
性相 任意
你的圣物变为一个智能物品。它的阵营在阵营轴上和你的阵营偏差不会超过1阶。它获得阵营对应的特征。该圣物拥有30尺精确视觉和30尺模糊听觉。它的察觉受训，当你提高察觉熟练度时，它也会同时提高。选择它是否可以大声说话，或者只能和你进行心灵感应。它会获得和它背景故事相配的语言，通常是从你已知的语言中选择。圣物还可以选择三种技能受训，并且每次获得新赋礼的时候都能获得一次技能提升。你的圣物的心智属性分别为+3、+2和+1，由你选择分配。圣物还在意志豁免上受训，并且当你的意志熟练度提升时，它也会一同提升。每次获得新赋礼时，选择圣物其中一项心智属性提升2点，或者两项心智属性提升1点。圣物在其他方面和智能物品相同。


套装圣物 Set Relics
有些魔法物品彼此非常协调，形成一个圣物种子，这叫做套装圣物或物品套装。这些物品符合特定主题，能赋予特定类型的角色力量。套装圣物基于套装内的物品生效，会变得更加强大。

战役中的套装圣物 SET RELICS IN THE CAMPAIGN
套装圣物比一般的圣物种子更强大，但它们也能让获取宝藏变得更加让人兴奋。为了保证平衡，确保每个角色都能解除到适合他们能力的圣物。此外，考虑一下角色获取这种相当强大的力量需要什么。寻找物品或材料来制作他们肯定会带来特定的任务和挑战，甚至一整场冒险。强烈建议考虑调整财富（《游戏主持人指南》95页）来解释这些有价值的物品。

启动套装圣物 STARTING A SET RELIC
当角色从套装中获得一件物品（通常是最低级的物品）时，它将启动一套圣物。（你可以让角色从不同的物品开始。）这个物品和其他圣物一样，都有一个次等赋礼和两个性相。然后，就圣物的其他方面而言，该套装被视为一个单独的圣物运作。

利用套装圣物 UTILIZING A SET RELIC
玩家必须携带或穿戴套装内的物品才能获得相应的套装效果。任何套装圣物的持有者未携带或未穿戴的物品、任何放在异次元空间内的物品，都不计入套装圣物授予赋礼时所需要的物品数量。角色还必须绑定那些有绑定特征的物品，否则同样不计入套装圣物的数量。因此，如果玩家丢失了某件物品，而该物品属于套装圣物的一部分，那么该物品给套装圣物带来的任何好处都将失去，直到该物品被找回，必要时还需要绑定，并再次携带或穿戴。

提升套装圣物 ADVANCING A SET RELIC
角色每收集到圣物的一部分，都能进一步增强圣物的赋礼，并解锁额外能力。

赋礼Gifts：套装生物根据角色拥有的套装数量来授予赋礼，这需要角色满足赋礼的最小等级限制。这个进程和《游戏主持人指南》表格2-22：圣物赋礼（《游戏主持人指南》95页）中的赋礼数量、最小等级、赋礼类型和价格相对应。每个已知的套装圣物（见下文）在列出赋礼之前都会有一个数字，这个数字和表格2-22中的赋礼数量对应，也和套装圣物所需要的的物品数量相对应。

集合能力 Collective Features: 圣物还会授予使用者能力，提升圣物赋礼效果，这取决于生效的套装数量。这些能力是累加的，举例来说，一旦角色获得套装中的4件物品，那么他将拥同时有2件套、3件套、4件套的能力。每个已知的合集圣物都列出了不同套装数量下对应的能力。

小型套装 Smaller Sets: 对于少于五件物品的套装圣物，该圣物也允许持用者使用更强大的赋礼。套装中的一件物品在授予赋礼时，必须不低于表格2-22中授予对应赋礼所需的最低等级。为了满足这一要求，持用者可能需要将现有物品提升到合适的等级，或者找一个等级合适的新物品。

强化套装物品 Improving Set Items: 套装圣物中的物品并不是总是一成不变。它可以根据某一类物品的规则，使用符文或更高级版本进行改进。如果套装圣物在列出某件物品时标明了高等版本，那你就必须拥有该物品的高等版本，并且将其作为套装圣物的一部分时，像通常那样来获得好处。


已知套装圣物 KNOWN SET RELICS
已知存在几件套装圣物。你可以用合适玩家角色的物品来调整他们，或者根据这里给出的效果，创建自定义套装。这些套装圣物使用本书或游戏主持人指南中的性相。
