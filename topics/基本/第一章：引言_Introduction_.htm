开拓者特工补充专长Pathfinder Agent Feats


拥有开拓者特工入门专长（Pathfinder Agent Dedication，《预言破灭之年世界指南》23页）的角色可以选择以下专长作为补充。


导航术 DEAD RECKONING [AA]                   专长4

[变体] [专注] [技能]
先决条件 开拓者特工入门（Pathfinder Agent Dedication）, 生存受训
你经过历练，在任何时候都能保持极强的方向感，即便你不能使用罗盘或观察星空。你能凭直觉知道哪个方向是北（如果你所在的地方存在北这个概念的话）。


环境探险家 ENVIRONMENTAL EXPLORER             专长4

[变体]
先决条件 开拓者特工入门（Pathfinder Agent Dedication）
你已经穿越了足够多的荒野，对这些地方的危险有直观的了解。你的见识广博意味着你有时候会在还未尝试时就预见危险。即便你没有在探索模式中进行搜寻（Searching）行动，你也可以尝试寻找通常需要搜索才能找到的环境危害。你依然需要满足能找到某个特定危害的其他要求。


敏锐探险家 OBSERVANT EXPLORER                专长4

[变体]
先决条件 开拓者特工入门（Pathfinder Agent Dedication）
归功于你花时间在荒野环境中漫游，已经获得了对生物的直觉。有时候你会在附近的生物发起攻击前就注意到它们的存在，或者即便它们并不打算攻击你，你也能注意到它们的存在。即便你没有主动搜寻（Searching）它们，你也可以尝试一次检定来注意到那些试图对你进行避人耳目（Avoid Notice）动作的生物。你依然需要满足能找到某个特定生物的其他要求。


想你所学 REMEMBER YOUR TRAINING [A]             专长4

[变体]
先决条件 开拓者特工入门（Pathfinder Agent Dedication）
频率 每日1次
你在成为开拓者特工的训练中学习了各种你可能会遇到的生物，你能在需要的时候回忆那段训练来获得关键的洞察力。你针对你能看见的生物尝试一次回忆知识，如果你相关技能未受训，则使用你的等级作为熟练度加值（而不是0）。如果你在此检定中掷出失败（但不是大失败），则你自动成功。


宗教探险家 SPIRITUAL EXPLORER                   专长4

[变体]
先决条件 开拓者特工入门（Pathfinder Agent Dedication）
你深知，游荡的鬼魂对于不谨慎的冒险者来说是危险的 ，而你对它们尤其得心应手。即便你没有在探索模式中进行搜寻（Searching）行动，你也可以尝试寻找通常需要搜索才能找到的作祟。你依然需要满足能找到某个特定作祟的其他要求。


学以致用 WATCH AND LEARN [R]                   专长4

[变体]
先决条件 开拓者特工入门（Pathfinder Agent Dedication）
触发 一个你能看见的盟友，在一次基于力量或敏捷的技能检定中成功或大成功，并且该技能你未受训
团队中的探索者特工们往往技能互补。你在跟随专业老手完成任务的过程中也在不断练习。在接下来的1分钟内，你在观察的技能的检定中，可以用你的等级作为熟练度加值，即便你未受训。这并不能允许你尝试那些需要受训才能使用的技能动作，除非你可以如此做。


考古学家的警告ARCHAEOLOGIST’S WARNING [F]       专长6

[变体]
先决条件 开拓者特工入门（Pathfinder Agent Dedication）
触发 你即将以察觉、社群或贼活检定进行先攻
你用可见的或可听到的方式向通报警告危险，使它们的先攻骰获得+1环境加值。根据你采取手势还是大声呼叫，该自由动作获得视觉或听觉特征。


伪造赝品 CRAFT FACSIMILE                         专长6

[变体] [技能]
先决条件 手艺专家，开拓者特工入门（Pathfinder Agent Dedication）
你因为能找回重要的遗物而出名，无论这些东西来自废弃已久的神庙还是小偷的老巢，有时候你需要创造一个赝品来替换它。你可以花费1分钟时间，利用可用的材料拼凑出一件你可以看见的、负载不超过1的物品的赝品。赝品和正品有一致的围观和负载，如果别人没有仔细检查的话，会被当做正品。熟悉正品的生物在检查赝品时能自动发现它是赝品。一个不熟悉正品的生物仍然可以分辨出它是某种匆忙拼凑的东西，但他们可能无法得知正品本该是什么样子。


触类旁通 DIVERSE RECOGNITION [F]        专长8
[变体] [技能]
先决条件 开拓者特工入门（Pathfinder Agent Dedication），在某个可以用于回忆知识的技能中达到大师熟练度
频率 每轮1次
触发 你成功对一个你能看见的生物回忆知识，并且你所使用的技能具有大师熟练度
你的开拓者特工训练允许你快速评估多个威胁。立即使用同样的技能，对你能看见的另一个不同种类的生物尝试一次回忆知识。你依然必须满足该检定的需求才能如此尝试。


振作起来！SNAP OUT OF IT! [AA]          专长8

[变体] [技能]
先决条件 医疗大师，开拓者特工入门（Pathfinder Agent Dedication）
开拓者特工的训练让他们能照顾彼此的安全，你知道如何让你的盟友恢复理智。尝试对一个与你相邻的，且被迷魂、惊惧、震慑或呆滞的盟友进行一次医疗检定。你选择其中一种状态。DC为造成这种状态的效果DC。
大成功 你所选择的状态值减少2。如果你选择迷魂，则迷魂结束。
成功 你所选择的状态值减少1。如果你选择迷魂，则迷魂结束。
失败 目标不受影响。
大失败 你所选择的状态值增加1。如果你选择迷魂，则迷魂持续时间增加1轮。


寻路仪共振渗透者 WAYFINDER RESONANCE INFILTRATOR 专长8

[变体]
先决条件 开拓者特工入门（Pathfinder Agent Dedication）, 寻路仪共振修补匠（Wayfinder Resonance Tinkerer）
你善于利用你的寻路仪不同寻常的共鸣来帮助你进行欺骗。每日1次，你能启动你的寻路仪，以奥术法术的方式施放幻象伪装（illusory disguise），将其升阶为2环。此外，每日1次，你能以一个单独的想象动作启动寻路仪，将其变成另一个同样大小的其他组织成员或官员的徽章，例如蛇徽财团（Aspis Consortium）徽章。此变形持续1小时。


过目不忘 EIDETIC MEMORIZATION [AAA]              专长10

[变体]
先决条件 开拓者特工入门（Pathfinder Agent Dedication）
开拓者们常常需要在不够理想的情况下研究复杂的古代历史，你已经训练了自己的大脑，让其能充分利用有限的时间。你能快速记住一篇碑文、卷轴或其他你能清楚地看见的、长度不超过一页的文字。然后，一旦到了安全的地方，你就可以使用书写工具、涂蜡书板或其他书写介质准确地再现这些文字。你在复制它的时候并不需要理解或辨识出具体是什么语言。你的记忆能一直持续到你下次进行日常准备；此时如果你还未以书写或其他方式记录下你所看到的东西，你就会忘记它，任何使其重现的尝试最后得到的都只能是一段混乱且不可理解的文字。你一次只能记住一篇文本；每当你再次使用此能力时，之前的记忆就会忘却。


交换绑定 SWAP INVESTMENT          专长12

[变体]
先决条件 开拓者特工入门（Pathfinder Agent Dedication）
频率 每日1次
作为开拓者，你学会了如何立即使用你在现场所发现的资源。你绑定一个新的魔法物品，移除一个之前绑定的魔法物品。之前的物品不再计入你10件绑定物品的上限中。

贾妮拉的装备贴士 JANIRA’S TOP GEAR TIPS
以下这些装备是每个开拓者都应该准备的。
粉笔！记得花几个铜板买粉笔。你能用它在去过的地方做记号，或者在解谜的时候打草稿，或者用来和他人交流。你甚至可以掰几个粉笔头下来，在必要的时候扔给别人。
别忘了镜子！镜子可以用来隔着墙角窥视，或者改变光线的方向。在紧急情况下，你甚至可以砸碎它来获得一把临时小刀。
绳子永远是你的朋友！你永远不知道你在什么时候需要把什么东西捆起来或保护自己，或者你可能只是需要一些够长的东西来看看走廊尽头有没有陷阱。
