休整期模式

休整模式 Downtime Mode

休整模式是逐日推进的，而非按照分钟或场景为单位。通常情况下此模式用于重现你身处于安全定居点时的环境，你可能正在一场冒险之后休养生息，或者研究在冒险中找到的神器。
休整模式能让你有时间进行充分休息，从事手艺或在其他专业上投入精力、学习新法术、重训专长或仅仅为了玩乐。你能够卖掉在冒险中获取的物品，购买补给，或者如你的专长、你的技能以及休整模式期间你所身处的定居点上所标注的那样执行其他行动。

长期休息 Long-Term Rest

在休整期间，你能花一天一夜的的时间进行休息来恢复生命值（Hit Points），恢复的数值为你的体质调整值（最低1点）乘以你的等级的两倍。

重训 Retraining

重训提供了一种调整角色选择的方式，当你想要把角色引导至新的方向或者认为之前作出的选择未能达到自身期待时十分有用。你能重训专长（feats）、技能（skill）以及部分可选的职业特性（class features）。你无法重训角色的族裔（ancestry）、传承（heritage）、背景（background）、职业（class）或属性（ability scores）。你无法在重训时执行其他休整期行动。
　　重训通常需要你耗费时间去向老师学习，无论是进行体能训练、在图书馆学习、还是一同陷入神游的魔法冥想。由你的GM决定是否能够得到合适的训练或者某些内容是否能够被重训。在某些情况下，你需要向指导者支付佣金。
　　有些能力可能很难甚至无法被重训（举例说明，术士只有在极端特殊的情况下才能够重训血统）。
　　在重训中选择想要重训的原选项时，你会经常遇到无法选择该选项（进行重训）的情况。举例说明，你不能把2级技能专长和4级的技能专长交换，或者你选择的原专长在被获取时还未满足某个选项所要求的先决条件。若你不记得想要重训的某个选项在原本选取时是否满足新选项的先决条件，则可以询问GM让他决定。若由于你进行重训而导致角色不再满足某个能力的先决条件的话，你就无法在此情况下使用该能力。为了获取所有你想要的新能力，你可能需要逐次依序重训数个能力才能达成。

专长 Feats

你能花费1星期的休整期来重训，用于更换角色的1个专长。移除旧专长并用相同类型的其他专长进行替换。举例，你能将技能专长更换为另一个技能专长，但是无法把它换成法师专长。

技能 Skills

你能花费1星期的休整期来重训，用于更换角色的1次技能提升。让失去技能提升的技能熟练等级下降一级，并让另一个技能的熟练等级提升一级。新熟练等级必须与你转移的熟练度相同或更低。举例说明：若你的吟游诗人在表演和隐秘上都是大师熟练度，神秘是专家，你能够将隐秘的熟练度降到专家，让神秘变为大师，但是你不能把技能提升分配到表演上把表演变为传奇熟练度。当你重新分配技能提升时请注意角色的等级；你的技能熟练度发生改变的等级会影响‘重训带有技能熟练度的专长’的能力。
　　你还能够花费1星期时间来重训一项在创建角色期间获得的初始受训技能（initial trained skill）。

职业特性 Class Features

你能够重训需要进行选择的职业特性，用不同的选项来替换它。一些选择，像是改变你法术库中的一个法术，需要1个星期。重训更大些的选择，像是德鲁伊教团或奥术学派，则由GM告诉你这到底会花费多久的时间——通常来说要用1个月。

其他休整期行动 Other Downtime Activities

如果你有其他想用在休整模式上的想法，请和你的GM进行沟通。你可能需要支付生活开支（具体价格见294页）。你可以获得财产，经营生意，成为行会或者市民团体的一员，在大城市巴结高层，指挥军队，收徒弟，成家或者成为引导一群信徒的牧师。

休整期技能行动 Skill Downtime Activities

第四章：技能介绍了数个休整期行动，此段落对这些行动进行了总结。
　　制作（Craft）：使用手艺技能，你能够使用原材料制作物品（见第四章244页的手艺技能）。
　　伪造文书（Create Forgery）：你伪造了一份文件（见第四章251页的社群技能）。
　　赚取收入（Earn Income）：你赚取金钱，这通常会使用手艺、学识或表演技能（见第四章236页的通用技能动作）。
　　求生（Subsist）：你再荒野中或者在定居点内寻找食物和庇护所（见第四章见240页的通用技能动作）。
　　治疗疾病（Treat Disease）：你花时间照顾一名染病的生物并希望将其治愈（见第四章248页的医疗技能）。
