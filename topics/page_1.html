第三章：职业
就像你角色的族裔在表现他们的身份和世界观起到的关键作用一样，他们的职业代表着他们曾经受到的训练以及作为一名冒险家的精进方向。为你的角色选择一个职业或许是你将为他们做出的最重要的决定。组成一个队伍的玩家们常常会创建那些在彼此的技能和能力上机械互补的角色——例如，确保你的队伍里包含一名治疗者，一名专注战斗的角色，一名擅长隐匿的角色，以及一些统御魔法的角色——所以你或许会希望在决定你的选项之前与你的队友们进行讨论。
每个职业下属的项目允许你实现丰富的角色概念。或许你想要创造一名才华横溢却缺乏专注的炼金术士，他能够快速背诵复杂的炼金物品公式，却很难记住他最好的朋友的生日。又或许你希望你的角色是一名肌肉结实的女剑士，在她举起盾牌的时候就变得不动如山。他们也可能是一名急躁的术士，总是用他因为天使血脉的传承而散发光芒的手指指指点点。你为你的角色在他们的职业项目中所做的选择——例如，为牧师选择神祇，为战士选择武器，或是为术士选择血脉——能够在这个规则与世界所建立的环境中实现你的想象。
　　本页后述的条目描述了Pathfinder中的12个核心职业。每个条目都包含了你扮演这个职业的角色，以及你将他们从初出茅庐的1级角色快速获取力量升至20级所需要知道的信息。除了职业条目外，你还需要参考后述的各个段落，它们详述了额外的角色选项以及如何提升你的角色等级。
  • 31页的提升等级Leveling Up将会告诉你如何在你得到足以提升新等级的经验值后将你的角色变得更强。
  • 214页的动物伙伴与魔宠Animal Companions and Familiars将提供创建一只和你共同冒险的动物伙伴或魔宠的规则。你必须拥有授予你伙伴或魔宠的职业特性或专长才能使用这些规则。
  • 219页的变体Archetypes将列出允许你更进一步定制角色能力的主题选项。虽然这些规则并不推荐初学者使用，不过这本书中的变体允许你从2级开始获取其他职业的能力。

阅读职业条目 Reading Class Entries

每个职业条目包括该职业典型成员的信息，外加关于扮演该职业的角色以及在多个游戏模式中游玩这些角色的建议。每个职业都会为你的角色提供属性提升和关键属性值；他们在每个等级获得的生命值数额；在多种能力、装备和技能上的熟练等级；来自职业特性的特殊能力；以及更多。角色的职业条目也会提供当他们升级时所需的信息，因此它将会是贯穿你整场战役的至关重要的参考资料。

扮演你的职业 Playing the Class

每个职业的第一个部分描述了该职业典型的核心利益与趋向性，以及外人如何看待他们的信息。这有助于激发你灵感以便决定你角色的行动以及定义他们的个性，不过这并不强迫你按照该部分的描述游玩你的角色。

关键属性 Key Ability

这是你所选职业的成员最需要考虑的属性值。你大多数最为有用和强力的能力都在某种程度上与这个属性相关联。
　　举例而言，你将使用这个属性决定与你的职业特性和专长相关联的困难等级（DC）。这被称作你的职业DC。如果你的角色是施法者职业的一员，这个关键属性还将用于计算法术DC和类似的数据。
　　大多数职业都与一种关键属性值相联系，但某些职业允许你从两种选项中进行选择。举例而言，如果你是一名战士，你能够选择力量或敏捷作为你的关键属性。选择力量的战士将更擅长白刃战，而选择敏捷的战士则更愿意从远处攻击或使用娴熟武器。
　　此外，当你选择角色的职业时，他们还会在关键属性值上获得属性提升，将关键属性值提高2点。更多关于属性提升的信息，请看20页。

生命值 Hit Points

这个部分将告诉你角色在每个等级从他们的职业那里获得多少生命值。将你在选择族裔时获得的生命值和这个条目中列出的数额，也就是一个固定值加上你的体质调整值，加在一起，以决定你角色的初始生命值。那些会让角色冲锋陷阵，短兵相接的职业将会在每个等级获得更高的生命值，而那些会让角色施展法术或擅长诡计的职业则给予较少的生命值。
　　每当你的角色获得一个等级，他们将提升等同于这个条目中列出的数额的最大生命值。更多关于计算角色体质调整值和决定他们生命值的信息，请看26页。

初始熟练 Initial Proficiencies

当你为你的角色选择职业时，他们获得一组初始熟练。熟练衡量了你角色执行任务，运用能力和成功进行检定的水平。熟练的等级范围从受训至传奇。举例而言，一名在长弓上受训的角色能够有效地使用它，而一名在这把武器上达到传奇的人则有可能做到百步穿杨！
　　每个职业条目都会指定你的角色在察觉，豁免，攻击，防御，以及法术或职业DC上的熟练等级。你能够在至少一个对你的职业来说比较重要的技能上获得受训熟练等级，你还能够选择其他的技能获得受训熟练——具体的数量由你的职业决定。如果你的职业让你在一个你已经受训的技能（通常是来自于你的背景）上再度受训，你可以选择将另一个技能变为受训。
　　熟练等级能够解锁多种专长和职业特性，它也有助于决定你所进行的任何检定的调整值或计算与那些项目相关联的DC。如果你的角色在察觉，一个豁免，或其他项目上受训，他们获得的熟练加值等同于他们的等级+2，而如果他们拥有专家熟练，他们获得的熟练加值等同于他们的等级+4。更多关于熟练等级的信息，请看13页。
　　施法职业获得对于法术攻击和DC的熟练等级，这会在每个职业的条目中更进一步详述。
　　如果某些事项没有在你的角色条目中列出，那么他们在这个项目上的熟练等级为未受训，除非他们通过其他来源获得了受训等级。如果你的角色在某件事上未受训，那么在尝试与这个项目相关联的检定或计算与之相关的DC时，将熟练加值视作+0。

升级表格 Advancement Table

这个表格概括了专长，技能提升，属性提升，以及你的角色因为等级提升而获得的其他好处。职业表格的第一列表明职业等级，而第二列列举了角色抵达这个等级时获得的每一项特性。1级的条目还会提示你选择你的族裔和背景。

职业特性 Class Features

这一部分呈现了职业授予你角色的所有能力。一个在高等级时获得的能力会在能力名旁边列出所需等级。所有职业都包含下面详述的这些职业特性，每个职业还会获得专属于它的特殊职业特性。许多职业特性要求在两个选项中做出选择。除非特定能力中声明，否则这些决定都不能在没有重训（如同481页解释的那样）的情况下改变。

职业专长 Class Feats

这个部分指定了你的角色获得的职业专长的等级——这是只有那个职业的成员才有权选择的特殊专长。职业专长最初在1级或2级时获得，根据职业而有所不同。特定职业专长将在每个职业条目的末尾详述。

技能专长 Skill Feats

这个部分指定了你的角色获得带有技能特征的专长的等级，这些专长也被称作技能专长。技能专长能够在第五章：专长中找到，这一章节开始于254页。在2级以及之后每2级，大多数职业都能获得一个技能专长，尽管游荡者Rogue能够更容易也更频繁地获取它们。你的角色必须在相应的技能上受训才能获得技能专长。

通用专长 General Feats

这个部分指定了你的角色获得通用专长的等级。大多数职业都能在3级和之后每4级时获得一个通用专长。每到这些等级时，你能够选择任何一个通用专长（包括技能专长），只要你的角色满足专长的先决条件。更多的信息能够在第五章：专长（254页）中找到。

技能提升 Skill Increases

这个部分指定了你的角色能够在某个技能的熟练等级上获得提升的等级。在3级和之后每2级时，大多数职业获得一个技能提升，尽管游荡者能够更容易也更频繁地获取它们。你的角色能够使用技能提升来在一个他们没有受训的技能上变得受训，或是在一个他们已经受训的技能上成为专家。
　　如果你的角色至少有7级，他们能够使用技能提升来在一个他们已经是专家的技能上成为大师。如果他们至少有15级，他们能够使用技能提升来在一个他们已经是大师的技能上成为传奇。

属性提升 Ability Boosts

在5级和之后每5级，你的角色能够提升四项不同的属性值。你的角色能够使用这些属性提升将他们的属性值提升至18以上。提升一个已经有18或更高的属性值意味着将它的数值增加1，而提升低于18的属性值则将数值增加2。更多关于属性提升的信息和在角色创建中运用它们的方法，请看20页。

族裔专长 Ancestry Feats

这个部分用于提示你的角色将在5级，9级，13级和17级时获得族裔专长。族裔专长在第二章的每个种族条目下详述，这一章节开始于32页。
