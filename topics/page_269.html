先知

先知 Oracle

神秘力量赋予你神力和诅咒。
先知入门 Oracle Dedication专长2
变体入门兼职
先决条件 魅力14
选择1个秘示域（72页）。你在宗教以及与秘示域相关的技能上获得受训熟练度；若你已经达到受训熟练度，则自行选择1个技能获得受训熟练度作为替代。你获得秘示域诅咒一栏内在第一段描述中最轻微、且常驻的效果，但你无法获得秘示域通常会提供的其他效果。你能像先知一般施法，并且会获得施法行动。你获得具有2个戏法的法术库，这可以是常见神术戏法，也可以是你领悟或学会的其他神术戏法。你在神术法术攻击骰和神术法术DC提升至受训熟练度。你在施放先知变体法术时的关键施法属性为魅力，且这些法术为神术先知法术。
　　特殊：直至选取另外2个先知变体的专长之前，你无法再选择任何其他入门专长。
基础秘术 Basic Mysteries专长4
变体
先决条件 先知入门
你获得1个自选的1级或2级先知专长。
基础先知施法 Basic Oracle Spellcasting专长4
变体
先决条件 先知入门
你获得基础施法能力（150页）。每当你通过先知变体而获得新环级的法术位时，将1个此环级的常见神术法术、或者你领悟或学会的其他神术法术加入你的法术库。
最初启示 First Revelation专长4
变体
先决条件 先知入门
你获得自身秘示域的初始启示法术。施放该法术会增强你的诅咒。使用你的秘示域的低阶诅咒，但是当你再次增强诅咒时，你不会获得中阶诅咒的效果，取而代之的是你会在低阶诅咒的效果之外额外陷入措手不及状态。
　　若你没有聚能池，那么你会获得具有1点聚能点的聚能池。你能够通过调和体内冲突的神力本质来进行再聚能，而这也会降低你的诅咒程度。
进阶秘术 Advanced Mysteries专长6
变体
先决条件 基础秘术
你获得1个先知专长。在判断专长的先决条件时，你的先知等级等同于自身角色等级的一半。
　　特殊：你可以多次选取该专长。每次选取都会获得另一个先知专长。
神力赐福 Mysterious Breadth专长8
变体
先决条件 基础先知施法
除了最高的2个环级之外，每个先知变体专长提供的法术环级均会增加1个法术位。
专家先知施法 Expert Oracle Spellcasting专长12
变体
先决条件 基础先知施法，宗教达到大师熟练度
你获得专家施法能力（150页）。
大师先知施法 Master Oracle Spellcasting专长18
变体
先决条件 基础先知施法，宗教达到传奇熟练度
你获得大师施法能力（150页）。
