珍贵材料护甲
珍贵材料制作的护甲更为昂贵而具有特效。皮甲可以改用龙革，木甲使用暗木，金属护甲则用除暗木外的任何珍贵材料。因护甲的负载在穿戴时会减少，决定其材料价格时使用其携带负载值（材料章节见577页）。
精金护甲 Adamantine Armor物品12+
罕见
用法 穿戴护甲
负载 见具体护甲
精金护甲黑光锃亮，异常坚韧耐用。
类型：标准纯度精金护甲；等级：12；价格：1,600gp + 160gp每B；制造需求：原材料所含精金至少需要价值200gp + 20gp每B。
类型：高纯度精金护甲；等级：19；价格：32,000gp + 3200gp每B；制造需求：原材料所含精金至少需要价值16,000gp + 1,600gp每B。
寒铁护甲 Cold Iron Armor物品5+
用法 穿戴护甲
负载 见具体护甲
寒铁护甲能弱化触碰它的特定生物。拥有寒铁弱点的生物（通常是恶魔和精类）对穿戴寒铁护甲的单位无武装攻击大失败时，会获得恶心1。拥有寒铁弱点的生物穿戴寒铁护甲时会保持恶心1状态。
类型：低纯度寒铁护甲；等级：5；价格：140gp + 14gp每B；制造需求：原材料所含寒铁至少需要价值70sp + 7sp每B。
类型：标准纯度寒铁护甲；等级：11；价格：1,200gp + 120gp每B；制造需求：原材料所含寒铁至少需要价值150gp + 15gp每B。
类型：高纯度寒铁护甲；等级：18；价格：20,000gp + 2,000gp每B；制造需求：原材料所含寒铁至少需要价值10,000gp + 1,000gp每B。
暗木护甲 Darkwood Armor物品12+
罕见
用法 穿戴护甲
负载 见具体护甲
暗木护甲比通常的负载要降低1（若正常负载为1，则变为L，对于正常为L的护甲来说无效果）。它比正常木甲更容易穿戴，忽略护甲减值所需的力量值减少2，其移动速度减值减少5尺（本书中没有木甲这种类型的物品）。
类型：标准纯度暗木护甲；等级：12；价格：1,600gp + 160gp每B；制造需求：原材料所含暗木至少需要价值200gp + 20gp每B。
类型：高纯度暗木护甲；等级：19；价格：32,000gp + 3,200gp每B；制造需求：原材料所含暗木至少需要价值16,000gp + 1,600gp每B。
龙革护甲 Dragonhide Armor物品12+
罕见
用法 穿戴护甲
负载 见具体护甲
龙革护甲基于龙的类型免疫一种伤害类型（具体见579页）。穿戴龙革护甲会在对抗造成对应类型伤害的攻击和法术时，给AC和豁免提供+1环境加值。
类型：标准纯度龙革护甲；等级：12；价格：1,600gp + 160gp每B；制造需求：原材料所含龙革至少需要价值200gp + 20gp每B。
类型：高纯度龙革护甲；等级：19；价格：32,000gp + 3,200gp每B；制造需求：原材料所含龙革至少需要价值16,000gp + 1,600gp每B。
精灵链甲 Elven Chain物品13+
罕见
用法 穿戴护甲
负载 1
精灵链甲是用秘银制作的链甲衫，散发出微弱的光泽。它给AC提供+2物品加值且没有检定减值。
　　由精灵工匠运用古老的工艺制作而成，精灵链甲十分安静。不同其它链甲衫――即便是普通秘银链甲衫――精灵链甲没有嘈杂（Noisy）特征。该护甲可像其它秘银链甲衫一样蚀刻符文上去
类型：标准纯度精灵链甲；等级：13；价格：2,500gp；制造需求：原材料所含秘银至少需要价值3,125sp。
类型：高纯度精灵链甲；等级：20；价格：52,000gp；制造需求：原材料所含秘银至少需要价值26,000gp。

山铜护甲 Orichalcum Armor物品20
稀有
用法 穿戴护甲
负载 见具体护甲
因山铜其的魔力特性，山铜护甲最多可以蚀刻上4个性能符文而非3个。若你穿戴山铜护甲，它会增强你对未来的感知力，使你的先攻骰获得+1环境加值。
