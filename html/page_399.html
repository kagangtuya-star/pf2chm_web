元素论

元素论 Elementalism

世界即是元素位面的反映。火焰于地核熊熊燃烧，在土壤之中摇曳或熄灭。水流滋润了干燥的大地，使其更为柔软坚韧。而这一切都被大气笼罩其中，它为世界带来了生命。元素论相信这四种元素便是创造所有物质的成分。因而，魔法也只是在操作此四元素。而理解这些元素便是元素论的精髓。

德鲁伊派系 Druidic Orders

元素论者摒弃了传统的魔法划分方式，转而支持元素四分法，且很少会去接受正规的教育。相反，他们周游于格拉里昂，向环境与元素大师Elemantal Master们学习。这种旅行被称之为朝圣Pilgrimages，而他们在路上所学到的教训则被称之为揭露Exposures或形态Forms。
　　德鲁伊是最有名的一类元素大师，他们能够驾驭山岳或海洋中的自然之力。尽管他们有些成为了世界的守望者，但大多数的还是隶属于三种派系之一。在此些派系，以及专注于气的风暴派系（见CRB132），有相当数量的德鲁伊会浸淫于元素魔法，选择元素使Elementalist职业变体（206页）。无论如何，正如派系的德鲁伊能正常从原能法术表中获得法术一样，他们相信自己能以元素的角度获得更广大的视野来观察大自然。
　　元素派系的德鲁伊在有着深远元素传统的地区会更为常见，比如扎摩诔。

火焰派系 Fire Order

你与火焰有种亲和力，可以利用它们带来拯救或毁灭。你的特技技能上提升到受训熟练度。你同时获得火眼火肺Fire Lung德鲁伊专长。你获得野火Wildfire派系法术。放任非自然火灾蔓延或以破坏环境的方式阻止自然火灾是你的派系禁忌（这并不会禁止你使用火焰破坏或者强迫你去灭除人工或自然的火焰）。

岩石派系 Stoen Order

你如石头一般坚韧，无论其是自然或是人工的，你从它们的稳定存在中获得了舒适感。你的手艺技能提升到受训熟练度。你同时获得不移盘石Steadying Stone德鲁伊专长。你获得碎地Crushing Ground派系法术。毒害或污染土壤，不顾后果的开采大地以掠夺其自然资源是你的派系禁忌（这不会禁止你做负责任的挖掘或采矿行为）。

波涛派系 Wave Order

水乃是生命之源，而你已经学会了如何塑造流水。你的医疗技能提升到受训熟练度。你同时获得水滨快步Shore Step德鲁伊专长。你获得升浪Rising Surf派系法术。污染水源或放任污染水源者不受制裁是你的派系禁忌（这并不会强迫你对潜在的水污染危机采取行动或牺牲自己去对抗明显更强的敌人）。

四元素 The Four Elements

这些元素可以在自然世界被清晰地观察，但哲学家们将其归出更深层次的属性，那些可以在风中看到或在晶洞中听到的。
　　火 Fire有无限的潜力，但必须小心呵护。如果悉心照看，火焰会带来温暖和治愈。它们照亮黑暗，揭示真理，为新事物清除掉旧残余。但火也能带来燃烧、遮蔽和毁灭。一个元素论者必须学会积聚和点燃火焰。
　　土 Earth是摇篮，也是坟墓。是大多数生物的家，为它们提供营养和庇护。但是大地实在太过深厚与旁大，无法完全控制。一个元素论者必须仔细聆听石头的声音，否则可能会激怒它们。
　　水 Water舒缓且神秘的。大多数生物主要由水组成，需要水才能生存；因此，控制水赋予了元素论者控制生命流动的力量。虽然水是最容易被塑造的元素，但它的深度仍是个谜。
　　气 Air将生命呼入无数的世界，但它是如此短暂，常被忽视。事实上，气是最难处理的元素，众人都知道去学习掌所不可掌之物、见所不可见之物是有多么的困难。掌握气，需要直觉、敏锐和超凡的感知能力。

武僧架势 Monk Stance

武僧们操作周身的元素以显化出魔法。CRB已经提供了山岳架势和狂风架势来表现土与气元素。
波纹架势 Reflective Ripple Stance  专长1
塑能系武僧架势水
要求 你未穿甲
你进入一个流畅优雅的架势，你的动作和攻击都伴有少量流水。你可以进行流波攻击，造成1d6钝击伤害。它属于搏击武器组，具备灵巧、缴械、非致命、摔绊、无武装和水特征。
　　当你处于波纹架势时，你在卸武、游泳和摔绊的运动检定上获得+1环境加值，你在避免被卸武和摔绊的反射DC上获得+2环境加值。　　
　　特殊此专长获得神术或异能特征，以匹配你真气法术的根源，如果你有的话。
燃焰架势 Stoked Flame Stance  专长1
塑能系火焰武僧架势
要求 你未穿甲
你进入一个迅速热烈的架势。你可以进行烁焰攻击，造成1d8挥砍伤害。它属于搏击武器组，具备蓄力、非致命、横扫和无武装特征。如果你发动了烁焰攻击的重击专精效果，你可以使用另一种效果：若你的重击造成伤害，目标额外受到1d6持续火焰伤害。
　　在燃焰架势下，你的速度额外获得5尺状态加值。如果你拥有高速移动Incredible Movement特性，其加值提升至15尺，且在3级之后的每4级提升5尺。
　　特殊此专长获得神术或异能特征，以匹配你真气法术的根源，如果你有的话。
